KRYON | Tech Brand Identity System
A premium, scalable visual identity engineered for a cutting edge tech firm. This repository serves as the Single Source of Truth for the Kryon brand ecosystem.

🚀 Key Features
Visual Architecture: High contrast palette (#FF5F1F Electric Orange & #36454F Charcoal).

Scalable Geometry: Custom monogram and wordmark designed for 16px favicons up to large scale physical signage.

Design to Code: Comprehensive UI Kit components ready for front end implementation.

Documentation: 40+ page strategy guide covering typography, spacing, and brand voice.

📦 Deliverables
/assets: SVG/PNG Logo variants, Custom Iconography.

/style guide: PDF Brand Manual & Typography specs.

/mockups: High fidelity UI and Print visualizations.

Figma Adobe Illustrator InDesign SVG Optimization
